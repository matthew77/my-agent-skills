# My Agent Skills

A collection of OpenClaw agent skills developed for various integrations and use cases.

## Skill Collections

### Tavily Skills

Integration with Tavily's search and research APIs for AI applications.

- **tavily-best-practices** 📘 - Reference documentation for building production-ready Tavily integrations
- **tavily-search** 🔍 - Web search optimized for LLMs
- **tavily-extract** 📄 - Extract content from specific URLs
- **tavily-crawl** 🕷️ - Site-wide content extraction
- **tavily-research** 🧠 - AI-powered research with synthesis

**Installation:**
```bash
clawhub install tavily-best-practices
clawhub install tavily-search
clawhub install tavily-extract
clawhub install tavily-crawl
clawhub install tavily-research
```
